# Video tool

This project is based on two previous projects. Video-downloader and Video-editor. I merged them to one single video tool instead. 

## Downloader

Download videos using URL. Both single videos and playlists are supported.

## Editor

Now there is only clip tools added. FPS, CRF, Speed and Resolution is changeable at the moment. And you can concat several clips to one highlights video. You can also choose between CPU and GPU rendering. Only Nvidia is supported at the moment. More features are coming.
